# interactive-elo
A quick CLI to calculate ELO changes

## Usage
 - Run `bundle install` to install dependencies
 - Run `thor elo:calculate` and follow prompts

Example:
```
$ thor elo:calculate
Team 1 ELO before match:  1500
Team 2 ELO before match:  1500
Which team won, 1 or 2:  1
Team 1: 1512
Team 2: 1487
```
